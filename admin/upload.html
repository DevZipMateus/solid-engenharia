Gerenciar Imagens do Site

Selecione as imagens para cada seção e salve-as na pasta correta do projeto.

Como usar

1

Clique em "Selecionar imagem" no card de cada seção para escolher uma foto do seu computador.

2

Visualize o preview da imagem. Quando estiver satisfeito, copie o arquivo para a pasta assets/images/ do projeto com o nome indicado no card.

3

O site carregará a imagem automaticamente a partir do caminho assets/images/nome-do-arquivo.jpg.

4

Para a seção Hero, o arquivo deve se chamar hero-bg.jpg. Para o logo, logo.png (fundo transparente recomendado).
